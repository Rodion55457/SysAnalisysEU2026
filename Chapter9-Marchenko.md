# Cистемний аналіз та проектування інформаційних систем.
## Марченко Родіон 241/з

**Можливості масштабування системи:** 

Оскільки ІС є локальною для конкретного закладу, то межі маштабування будуть натурально обмежені розміром єдиного закладу. Проста клієнт-серверна архітектура легко дозволяє збільшувати кількість кінцевих точок доступу для користувачів, а робота системи через стандартну локальну мережу Ethernet робить налаштування комунікацій зручним, а самі комунікації легко модифікованими.
